# node_calculator:
To run the application:
npm start

The project uses several testing tools such as Jest, babel-jest, and @babel/preset-env. Here is a description of those test scripts:

To run all tests:
npm test
To run unit tests:
npm run test:unit
To run integration tests:
npm run test:integration
